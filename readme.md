# Blog Mgt API Backend

This is a simple RESTful API backend for a blogging application. The backend is built with Node.js, Express, and MongoDB, and it includes user authentication using JSON Web Tokens (JWT).

## Features

- User registration and login with hashed passwords.
- JWT-based authentication.
- CRUD operations for blog posts.
- Association between users and their blog posts.

## Technologies Used

- Node.js
- Express.js
- MongoDB with Mongoose
- JWT for authentication
- bcrypt for password hashing
